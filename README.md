# FranStore API

1. Clonar proyecto
2. Instalar las dependencias:

```
npm install
```

3. Copiar el archivo `.env.template` y renombrarlo a `.env`
4. Cambiar las variables de entorno
5. Levantar la base de datos:

```
docker compose up -d
```

6. Ejecutar el seed:

```
http://localhost:3000/api/seed
```

7. Correr el proyecto:

```
npm run start:dev
```
